# Sign Language Detection

This project detects hand gestures in real-time using **Python**, **OpenCV**, **MediaPipe**, and **cvzone**.  
It is used for creating datasets and identifying hand gesture shapes for Sign Language recognition.

---
 Features
- Real-time hand tracking  
- Cropping and resizing of hand region  
- White background normalization  
- Saves hand gesture images automatically  
- Uses MediaPipe + cvzone for accurate hand detection
